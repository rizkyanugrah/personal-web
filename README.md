## Getting Started

- Clone this repository

HTTPS:

```
git clone https://github.com/rizkyanugrah/personal-web.git
```

SSH:

```
git clone git@github.com:rizkyanugrah/personal-web.git
```

## After cloning, open file index.html into your browser

```
index.html
```
